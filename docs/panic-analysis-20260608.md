## 二、周一3种可能路径（策略不变）

### 🟢 路径①：低开恐慌→日内V反（概率≈50%）

9:30低开1-2%、恐慌盘涌出 → 10:00砸到低点 → 资金抄底 → 午后拉回

**操作：** 不开新仓，不急跌不卖，等午后确认。

低开 → 反弹无力 → 尾盘继续下杀收最低

**操作：** 减仓弱势股，执行止损。

### 🔵 路径③：小幅低开→快速修复（概率≈20%）

低开不到1%，政策/消息对冲直接拉红

**操作：** 持有不动。

> **核心：** 恐慌急跌时不要卖 — 等第一个5-15分钟走完看后续确认。

## 五、周一整体策略

### 开盘15分钟判断

| 观察点 | 判断 |
|:-------|:-----|
| 低开多少 | 1-2%正常，>3%算恐慌 |
| 前15分钟量 | 放量急跌→恐慌；缩量低开→已消化 |
| 能V回来吗 | 10:00前见低点后有反弹→V反；一路跌→持续弱 |

### 本次报告 vs 旧版修正要点

1. **腾景科技(688195)** — 原判定"下降趋势"，前复权修正后为"区间震荡·区间顶部"，策略从"关注趋势能不能撑住"改为"正常持有等反弹"
2. **金安国纪(002636)** — 上涨趋势确认，量比1.44放量，结构最好标的之一
3. **沪电股份(002463)** — 上涨趋势·上行 EMA5/EMA10 多头排列确认
4. 所有标的的信号检测基于修正后的EMA计算（close_slope改为3日），更灵敏捕捉趋势拐点

### 按走势分类

**路径①（V反，概率50%）**
→ 什么都不动。抗跌的标的优先反弹。

**路径②（持续弱，概率30%）**
→ 优先减仓接近止损的标的 > 超配标的各减部分。

**路径③（快速修复，概率20%）**
→ 什么都不动。

### 核心原则

- 上涨趋势的标的 → 恐慌是关注点，不是卖点
- 区间底部的标的 → 已经最低区域
- 接近止损的标的 → 到了就走，不犹豫
- 结构误判已修正的标的 → 修正判断，持有等反弹

> **简放：** 「降低止损的核心，不完全在于止损点的设置，而在于耐心等待一个好买点」
