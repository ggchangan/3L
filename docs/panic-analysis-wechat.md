## 二、周一3种可能路径

### 🟢 路径①：低开恐慌→日内V反（概率≈50%）

9:30 低开1-2%、恐慌盘涌出 → 10:00砸到低点 → 资金抄底 → 午后拉回

**操作：** 不开新仓，不急跌不卖，等午后确认。

### 🟡 路径②：低开震荡→持续走弱（概率≈30%）

低开 → 反弹无力 → 尾盘继续下杀收最低

**操作：** 减仓弱势股，执行止损。

### 🔵 路径③：小幅低开→快速修复（概率≈20%）

低开不到1%，政策/消息对冲直接拉红

**操作：** 持有不动。

> **核心：** 恐慌急跌时不要卖 — 等第一个5-15分钟走完看后续确认。V反→持有，持续弱→减仓，快速修复→不动。

→ 沪电股份、长川科技、长电科技

> **核心逻辑：** 恐慌企稳后，最先涨的不会是跌最惨的，而是本来就最强、结构没坏的。

## 四、个股案例分析

以下基于简放3L体系结构判定+信号检测，对市场关注的代表性标的进行逐一分析，标注操作策略。

### 🟢 光迅科技 （002281）— CPO

CPO概念周五逆势+2.55%，光迅自己+3.18%收红，最抗跌之一。结构区间震荡中段，没问题。

**周一应对：** 低开就拿着，CPO主线没变。如果低开超-5%到支撑位附近缩量企稳，反而是关注点。

### 🟡 长川科技 （300604）— 半导体设备

**周一应对：** ⚠️ 止损较近。如果周一低开后反弹无力，考虑减半。如果低开直接到支撑位附近观察是否缩量企稳。

## 五、周一整体策略

### 开盘15分钟判断

| 观察点 | 判断 |
|:-------|:-----|
| 低开多少 | 1-2%正常，>3%算恐慌 |
| 前15分钟量 | 放量急跌→恐慌；缩量低开→已消化 |
| 能V回来吗 | 10:00前见低点后有反弹→V反；一路跌→持续弱 |

### 按走势分类

**路径①（V反，概率50%）**
→ 什么都不动。抗跌的标的优先反弹。

**路径②（持续弱，概率30%）**
→ 优先减仓接近止损的标的 > 超配标的各减部分。

**路径③（快速修复，概率20%）**
→ 什么都不动。

### 核心原则

- 上涨趋势的标的 → 恐慌是关注点，不是卖点
- 区间底部的标的 → 已经最低区域
- 接近止损的标的 → 到了就走，不犹豫
- 突破被大盘拖回的标的 → 等两天看效力还在不在，不急
- 结构误判的标的 → 修正判断，持有等反弹

> **简放：** 「降低止损的核心，不完全在于止损点的设置，而在于耐心等待一个好买点」
