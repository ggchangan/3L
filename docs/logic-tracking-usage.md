## 1. 快速开始

### 1.1 进入页面

浏览器访问导航栏的 **💡 逻辑追踪** 或直接打开：

`/logic-tracking`

### 1.2 第一步：创建逻辑标签

点击右上角 **「+ 新建逻辑」**，填写：

| 字段 | 说明 | 示例 |
|:----|:-----|:-----|
| **ID** | 唯一标识，只能用英文 | `tag-ai-chain` |
| **名称** | 逻辑方向名称 | `AI算力链条` |
| **描述** | 简要说明（可选） | `英伟达链→光模块/PCB/封测` |
| **层级** | 聚焦/核心/观察 | `🌟 聚焦` |
| **关联行业** | 逗号分隔 | `光模块,PCB,服务器` |
| **关联个股** | 逗号分隔，6位代码 | `300502,002916,600584` |

**层级说明：**

| 层级 | 上限 | 特点 |
|:----|:----:|:-----|
| 🌟 **聚焦** | 最多3个 | 页面顶部突出显示，自动验证+异常推送 |
| 📌 **核心** | 不限 | 正常显示，自动验证不推送 |
| 👁️ **观察** | 不限 | 默认折叠显示，适合"先观察看看" |

> 聚焦满了3个再新增时会提示「聚焦层级最多3个」，需要先降级一个。

## 2. 管理逻辑标签

### 2.1 三级展示

页面按层级分成三个区块：

```
🌟 聚焦（2/3）          ← 顶部突出显示，彩色标签，最多3个
  ├── AI算力链条          ← 显示关联个股数、事件数、印证率
  └── 先进封装加速

📌 核心（3） [折叠]      ← 默认展开，可折叠
  ├── 光伏
  ├── 机器人
  └── 创新药

👁️ 观察（5） [折叠]      ← 默认折叠，展开后显示
  ├── 储能（还有3个...）
  └── 低空经济
```

### 2.2 操作

每张标签卡片右侧有 **编辑** 和 **删除** 按钮：
- **编辑**：修改名称、描述、层级、关联行业/个股
- **删除**：删除逻辑标签（确认后不可恢复）

## 3. 资料投喂

### 3.1 投喂方式

点击右上角 **「📤 投喂」**，输入文章链接：

```
[https://mp.weixin.qq.com/s/...           ] [提取]
```

支持微信公众号、雪球、财联社等主流文章链接。

### 3.2 处理流程

```
粘贴链接 → 点击「提取」
  ↓
系统自动：
  ① 抓取文章内容
  ② 提取标题+摘要
  ③ 打标签（关键词匹配 → 推荐匹配/LLM兜底）
  ↓
预览确认：
  ├── 标题：XXXXX
  ├── 来源：微信公众号
  ├── 摘要：文章核心内容...
  └── 匹配标签：
      ☑ AI算力链条 (99分)    ← 系统推荐，可修改
      ☐ 先进封装加速 (40分)
  ↓
确认保存 → 归入对应逻辑标签，事件数+1
```

### 3.3 注意

- **关键词匹配**：系统优先用关联行业/个股匹配，匹配度高的直接推荐
- **LLM匹配**：关键词匹配不明确时自动走AI语义匹配（需要配置LLM_API_KEY）
- **标签可修改**：保存前可以取消或勾选任意标签
- **无LLM不影响**：系统没有AI也能跑，只是标签推荐靠关键词匹配

## 4. 每日自动扫描

### 4.1 涨停扫描 🔴

每天收盘后自动拉取当日的涨停板池，扫描逻辑：

```
涨停股池（~50-100只/日）
  ├── 匹配已有逻辑 → 记录为验证通过事件
  │   ├── 1板：健康分+1
  │   ├── 2连板：健康分+2
  │   └── 3板+多只联动：健康分+3，推送提醒
  └── 未匹配 → 标记为候选（后续LLM自动提炼）
```

**效果示例（2026-05-26 真实数据）：**

```
涨停46只 → 匹配命中3只
✅ 华天科技(002185) 2连板 → 半导体国产替代 ✅
✅ 米奥会展(300795) 1板 → 会展经济 ✅
✅ 长电科技(600584) 2板 → 半导体国产替代 ✅
❌ 43只未匹配（新亚制程、三佳科技、香江控股...）
```

### 4.2 新高扫描 🔵

同涨停扫描，但信号更强——新高说明逻辑已被市场充分定价：

```
新高股池（~200-300只/日）
  ├── 匹配已有逻辑 → 健康分+3（新高信号比涨停可靠）
  └── 未匹配 → 标记为候选
```

**效果示例：**

```
新高294只 → 匹配命中2只
✅ 华天科技(002185) +10.02% → 半导体国产替代
✅ 米奥会展(300795) +19.98% → 会展经济
```

> 涨停是「早发现」，新高是「强验证」。两者结合，涨停发现的逻辑如果后续走新高，则信号确认。

## 6. 常见问题

### Q：聚焦只能3个吗？

对。"将军赶路，不追小兔"。3个聚焦方向已经很多了，强迫你聚焦在最强的逻辑上。

### Q：LLM打标签需要什么配置？

需要设置环境变量：
- `LLM_API_KEY` = 你的API密钥（DeepSeek等）
- `LLM_API_URL` = API地址（默认 https://api.deepseek.com/v1/chat/completions）
- `LLM_MODEL` = 模型名（默认 deepseek-chat）

不设置也不影响使用，系统会用关键词匹配。

### Q：涨停/新高扫描什么时候跑？

当前是手动触发（后续会加到17:00的数据更新cron里）。可以在终端运行：

```bash
python3 -c "from backend.services.logic_zt_scanner import scan_zt_pool; r=scan_zt_pool(); print(f'匹配 {len(r[\"matched\"])}/{r[\"total\"]}')"
python3 -c "from backend.services.logic_high_scanner import scan_new_highs; r=scan_new_highs(); print(f'匹配 {len(r[\"matched\"])}/{r[\"total\"]}')"
```

### Q：旧的数据会丢失吗？

所有数据存在 `data/logic_tracking.json`，原子写入（tmp+rename），不会因写入中断丢失。

### Q：可以在微信里用吗？

目前是Web页面，后续会考虑PWA/Capacitor打包。
