# 最强逻辑追踪系统 — 产品设计文档

> 版本：v0.4 | 日期：2026-05-27 | 状态：初稿待讨论
> 产品视角：以信息输入→自动处理→走势验证为核心链路

---

## 1. 核心理念

> **不是先想好逻辑再录入，而是从资料中自动提炼逻辑。**

现有系统的 L2（最强逻辑）是空白的。这套系统的定位是：**你负责读资料、判断逻辑，系统负责归类、跟踪、验证。**

---

## 3. 核心概念

### 3.1 逻辑标签

逻辑标签是整个系统的核心。它不是自动生成的，而是**你创建**的——你读研报/文章发现一个逻辑方向，就创建一个标签。

```
例子：
  标签名: "先进封装加速"
  创建时间: 2026-05-27
  关联行业: [半导体, 封测]
  关联公司: [长电科技, 通富微电, 华峰测控]
  状态: 活跃 | 验证中 | 淡化 | 失效
```

### 3.2 资料条目

每条你投喂的资料（文章/研报/视频/新闻）是一个条目：

```
📄 涛定律深度解读（公众号）
  ├── 摘要: "华为提出涛定律，推动先进封装技术变革..."
  ├── 行业标签: 半导体, 封测      ← 自动
  ├── 公司标签: 长电科技, 通富微电  ← 自动
  ├── 核心逻辑标签: 先进封装加速    ← 自动匹配 or 你手动指定
  ├── 原文链接: https://...
  └── 投喂时间: 2026-05-27 14:30
```

### 3.3 催化剂事件

资料条目本身就是一个催化剂事件。多个事件按时间排列在逻辑标签下面，构成时间线。

## 4. 三大功能模块

### 模块A：逻辑标签管理（P0）

| 功能 | 说明 |
|:----|:-----|
| 创建标签 | 输入名称+关联行业+关联公司+初始状态 |
| 编辑标签 | 修改名称/行业/公司/状态 |
| 删除标签 | 删除后，关联的资料不会消失，只是标签解除 |
| 标签列表 | 按状态筛选（活跃/验证中/淡化/失效） |

### 模块B：资料投喂与处理（P0 ~ P1）

| 功能 | 说明 |
|:----|:-----|
| 投喂资料 | 粘贴链接或直接输入文本内容 |
| 自动提取摘要 | 提取核心内容摘要 |
| 自动打行业/公司标签 | 根据内容匹配已有的行业和A股公司 |
| 自动匹配核心逻辑标签 | 将资料归类到已有逻辑，或提示新建标签 |
| 手动调整标签 | 自动打的不准，你可以改 |

#### 三种信息源处理方式

| 信息源 | 投喂方式 | 备注 |
|:------|:--------|:-----|
| 公众号文章 | 粘贴链接，系统抓取正文 | 大部分公众号可抓取 |
| 研报 | 粘贴链接或上传PDF摘要 | 整份研报处理可能较复杂，先做摘要 |
| 抖音/视频 | 粘贴链接+手动补充摘要 | 视频内容无法自动提取，需你补充 |
| 新闻链接 | 粘贴链接，系统抓取正文 | 同公众号 |

### 模块C：走势验证（P1 ~ P2）

| 功能 | 说明 |
|:----|:-----|
| 事件后走势展示 | 每条资料投喂后，展示3/5/10天关联个股和板块走势 |
| 走势评分 | 系统展示数据，**你手动打分**：印证/一般/背离 |
| 逻辑健康度总览 | 每个逻辑标签下，所有事件的平均印证率 |

## 7. 待讨论的问题

| # | 问题 | 结论 |
|:-:|:----|:----:|
| ① | 投喂方式 | **链接抓取 + PDF上传 + 抖音链接**（抖音提取程序已有，后续部署）|
| ② | 核心逻辑标签由谁打 | **系统自动提取 + 推荐已有标签 + 手动输入**，三种方式灵活配合 |
| ③ | 走势验证 | **手动模式**：投喂后点按钮触发，自动展示走势结果；**自动模式**：未来系统自动抓新闻跑验证并推送 |
| ④ | L3自动抓新闻 | ❌ 暂时不做 |
| ⑤ | 放哪个页面 | **独立页面**，导航栏加入口 |
| ⑥ | 数据存哪里 | `data/logic_tracking.json`，资料不丢靠：容器挂载卷 + 定时备份 + 收藏标记 |

## 8. 开发优先级（建议）

| 优先级 | 功能 | 说明 |
|:-----:|:----|:-----|
| **P0** | 逻辑标签 CRUD | 创建/编辑/删除/列表，存 JSON 文件 |
| **P0** | 投喂资料（手动输入摘要模式） | 粘贴链接+摘要，手动选标签 |
| **P1** | 自动提取摘要+打标签 | 从链接抓正文，AI 自动处理 |
| **P1** | 逻辑详情页 | 按标签聚合显示所有资料+走势验证 |
| **P1** | 走势验证数据展示 | 自动算关联个股涨跌+板块排名 |
| **P2** | 走势验证打分 | 你手动打分，系统记录历史 |
| **P2** | 催化剂时间线 | 按时间排列所有事件 |
| **P3** | 自动抓取相关新闻 | 主动搜索匹配逻辑标签的新闻 |
| **P3** | 知识库文章关联 | 已有的 knowledge_base 文章打上标签关联进来 |
