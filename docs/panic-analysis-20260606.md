<div class="path-card path-1">

<div class="path-card path-2">

## 五、周一整体策略

### 开盘15分钟判断

| 观察点 | 判断 |
|:-------|:-----|
| 低开多少 | 1-2%正常，>3%算恐慌 |
| 前15分钟量 | 放量急跌→恐慌；缩量低开→已消化 |
| 能V回来吗 | 10:00前见低点后有反弹→V反；一路跌→持续弱 |

### 按走势分类

**路径①（V反，概率50%）**
→ 什么都不动。金安国纪/沪电/雅克/光迅这些抗跌的优先反弹。

**路径②（持续弱，概率30%）**
→ 优先减仓：长川科技 ⚠️ > 通富微电 > 兴森科技 > 超配股（沪电/杰瑞/长电各减2%）

**路径③（快速修复，概率20%）**
→ 什么都不动。

### 核心原则

- 上涨趋势的**金安国纪/沪电股份** → 恐慌是关注点，不是卖点
- 区间底部的**智微智能/德福科技** → 已经最低区域
- 接近止损的**长川/通富/兴森** → 到了就走，不犹豫
- **中钨高新** → 等两天看突破效力还在不在，不急
- **腾景科技** → 修正为区间震荡，持有等反弹

> **简放：** 「降低止损的核心，不完全在于止损点的设置，而在于耐心等待一个好买点」
