# get_stage() 加速/滞涨/加速跌 实现记录（2026-05-21）

## 背景

`get_stage()` 原代码对非区间震荡只返回 上行/下行/转折 三种，但skill文档已有 加速/滞涨/加速跌 的描述。本次补全代码实现。

## 算法

取完整EMA10序列，取后半段（约30根）计算s1=前半段斜率, s2=后半段斜率：

```python
valid = [v for v in ema10 if v is not None]
half = len(valid) // 2
s1 = _reg_slope(valid[:half])
s2 = _reg_slope(valid[half:])
```

ratio = s2 / s1（当|s1| > 1e-8），否则ratio=1.0（平坦时归为普通阶段）。

## 判定表

### 上涨趋势（s1>0, s2>0）

| ratio | 阶段 | 含义 |
|-------|------|------|
| > 1.8 | 加速 🚀 | 后段斜率比前段陡1.8倍以上，需求快速消耗 |
| < 0.4 | 滞涨 ⚠️ | 后段斜率不到前段4成，上涨动力衰竭 |
| 0.4~1.8 | 上行 | 正常健康上涨，进二退一 |

### 下降趋势（s1<0, s2<0）

| ratio | 阶段 | 含义 |
|-------|------|------|
| > 1.8 | 加速跌 📉 | 下跌加速，供应急剧释放 |
| ≤ 1.8 | 下行 | 正常下跌 |

### 方向相反（s1×s2 < 0）

→ 转折（趋势转换中）
