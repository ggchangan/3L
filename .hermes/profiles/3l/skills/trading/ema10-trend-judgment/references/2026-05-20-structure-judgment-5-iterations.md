## 核心结论

### 结构判断 — 最终算法

```python
c15 = c60[-15:]
e10 = _ema_list(c15, 10)
n = len(e10)
max_pos = max(range(n), key=lambda i: e10[i])
min_pos = min(range(n), key=lambda i: e10[i])
first_q = n // 4
last_q = n - 1 - n // 4

if max_pos >= last_q and min_pos <= first_q:
    structure = '📈 上涨趋势'
elif min_pos >= last_q and max_pos <= first_q:
    structure = '📉 下降趋势'
else:
    structure = '➡ 区间震荡'
```

### EMA10滞后说明

EMA10是指数移动平均，天然滞后于实际价格2-3天。这是均线的正常特性，不是bug。当价格已经见顶回落但EMA10仍在爬升时，系统会暂时显示上涨趋势，直到EMA10方向跟随价格转变。
