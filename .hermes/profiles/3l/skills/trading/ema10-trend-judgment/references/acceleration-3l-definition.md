# 3L教材中的加速定义（6.5止盈点）

> 来源：3L交易体系构建 第6.5节止盈点 "加速"退出

## 核心定义

> **加速是定性而非定量** — "加速更多是定性，而非定量，或者说无法定量"

> 引《股票魔法师》：**"在1-3周内涨幅达到25%-50%或更高时，会达到巅峰的顶点"**

## 量价特征

1. **连续大阳线** — 推动股价快速上涨
2. **成交量明显放大** — "连续出现较大买盘"
3. **斜率变陡** — K线斜率在短时间内变得更加陡峭

> "加速的过程中，成交量明显放大，推动股价形成大阳线，这种行为快速消耗了需求，说明看好的人生怕错过，不计成本的买入了。"

## 供需格局解释

- 加速=快速消耗需求 → 站在场外有买入意愿的人少了
- 获利盘增加 → 兑现意愿的人增加了
- 结果：**该买的都买了，潜在的卖盘就大于买盘了**
- 加速后结果：剧烈波动或快速下跌

## 口诀

> **"加速无买点"** — 加速后一定不能追买

## 与EMA10斜率比法的关系

`ema10-trend-judgment` 的 `get_stage()` 中加速检测使用 **s2/s1 > 1.8**（s1=末15根EMA10整体斜率，s2=末3根斜率）：
- 这是EMA滞后后的量化近似，并非3L教材原文定义的加速
- EMA10滞后1-2天，所以EMA10斜率比法的加速信号可能**晚于**教材的K线形态加速
- 两种方法可以交叉验证：
  - K线形态法（见 `acceleration-judgment` skill）：看连续大阳+量放+涨幅扩大
  - EMA10斜率比法：看s2/s1 > 1.8

## 卖出决策

> "在加速形态出现后，就可以选择卖出，或者卖出一部分"

加速卖出是 **左侧止盈**（前置性卖点），配合右侧止盈（破关键点/破均线）使用：

```
加速信号 → 左侧卖一半
剩余半仓 → 等减速信号或反转信号再处理（右侧止盈）
```
