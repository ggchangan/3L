# 2026-05-21: EMA10结构判断bug修复

## 触发场景

用户为深圳华强(000062)的SVG图添加支撑/压力线时，发现结构被判定为「下降趋势」，但K线显示该股近期急跌后已在反弹，应属「区间震荡」。

## Bug定位

`ema_utils.py` 中 `get_structure()` 的代码：

```python
c15 = closes[-15:]
e10 = ema_list(c15, 10)          # ← BUG：只从15根K线算EMA10
```

EMA10在此处**没有继承历史数据**，起始值 = 第1根K线收盘价，纯靠15根数据衰减。

## 影响分析

**错误：** 任何15天前处于高位的股票，其EMA10起始值偏高，一路衰减到当前位置 → 最大值永远在第[0]位、最小值在第[14]位 → 满足「下降趋势」条件。

**误判范围：** 251只自选股中33只（13%）判断变化：

| 旧→新 | 数量 |
|:-----|:-----|
| 上涨趋势→区间震荡 | 10 |
| 下降趋势→区间震荡 | 5 |
| 区间震荡→上涨趋势 | 6 |
| 区间震荡→下降趋势 | 11 |
| 上涨趋势→下降趋势 | 1 |

**总趋势：** 修复后更多股票归为「区间震荡」（EMA10极值位置更合理）。

## 修复

```python
e10_full = ema_list(closes, 10)  # 全量计算
e10 = e10_full[-15:]              # 取末15根
```
