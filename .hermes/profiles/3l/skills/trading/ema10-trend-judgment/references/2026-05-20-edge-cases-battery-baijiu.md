## 案例二：白酒 — 孤立单日U错误判为区间震荡 ⚠️→✅

### EMA10线形态（15日）

```
[ 0] EMA10=2322  -
[ 1] EMA10=2315  ↓
[ 2] EMA10=2308  ↓
[ 3] EMA10=2306  ↓
[ 4] EMA10=2301  ↓
[ 5] EMA10=2294  ↓
[ 6] EMA10=2290  ↓
[ 7] EMA10=2288  ↓
[ 8] EMA10=2289  ↑   ← 孤立单日！+0.81点
[ 9] EMA10=2281  ↓
[10] EMA10=2270  ↓
[11] EMA10=2268  ↓
[12] EMA10=2255  ↓
[13] EMA10=2241  ↓
[14] EMA10=2230  ↓
```

- 方向序列：`DDDDDDD↑DDDDDD`
- 注意：只有一个孤立↑（第8天2287.82→2288.64，+0.04%）
- **连续段（≥2天）**：D段（7天）+ D段（6天）= 只有D段
- 孤立U段长度=1，＜2，被过滤
- 段间切换：0次
- **判定：📉 下降趋势 ✅**

### 关键教训

**单日反向波动不应计入方向切换。** 白酒的EMA10很清楚——整体趋势是下降的，第8天的一个微小涨幅（0.04%）只是EMA10对当日价格的自然反应，不是趋势反转信号。

**规律：** 连续≥2天同向才算一个有效段。孤立单日U/D是噪声，过滤掉。

| 特征 | 电池（真实切换） | 白酒（噪声） |
|:----|:--------------|:-----------|
| ↑段天数 | 10天 | 1天 |
| 是否连续≥2天 | ✅ | ❌ |
| ↓段天数 | 4天 | 7天+6天 |
| 总切换次数 | 1 | 0 |
| 正确判定 | 区间震荡 | 下降趋势 |

## 算法规则总表

```
1. 计算15日EMA10
2. 转方向符号（U/D）: e10[i] > e10[i-1] → U, < → D
3. 合并连续≥2天同向为有效段
4. 有效段之间做方向切换计数
5. 切换=0 → 趋势方向（看首段符号）
6. 切换≥1 → 区间震荡
```
